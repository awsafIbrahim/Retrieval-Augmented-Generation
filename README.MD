# Retrieval-Augmented Generation (RAG) System using Gemini API

A minimal implementation of a **Retrieval-Augmented Generation (RAG)** system built with **Python**, **FAISS**, and **Google Gemini API**.  
This project demonstrates how to combine local document retrieval with Geminiâ€™s language model to produce context-grounded answers from custom text data.

---

## Features
- Uses **Gemini text-embedding-004** for high-quality text embeddings  
- Builds a **FAISS** index for efficient vector similarity search  
- Retrieves top relevant text chunks for a given query  
- Generates contextual answers using **Gemini-2.0-Flash** or similar models  
- Fully modular and runnable in **VS Code** or **Google Colab**
